# MacroLab v2 architecture

## Platform layers
1. UI layer - homepage observatory and developer console
2. Data contract layer - typed schemas in `lib/schemas.ts`
3. Automation layer - Claude Skills workflows for shocks, regime, sectors, and insights
4. ETL layer - planned ingestion from global macro, Singapore macro, trade, rates, and geopolitical feeds

## Operating principle
The platform should explain how shocks move into Singapore, not only display raw indicator values.
